## Upgrade Instructions

Follow these steps to upgrade your Lawnstarter integration to the latest version.

<Steps>
  <Step title="Check Compatibility" icon="check-circle">

Review your current integration against the [migration guide](#).

  </Step>
  <Step title="Update SDK" icon="download">
  
<CodeGroup tabs="npm,yarn">
````bash
npm install lawnstarter-sdk@latest
````
````bash
yarn add lawnstarter-sdk@latest
````
</CodeGroup>

  </Step>
  <Step title="Update Configuration" icon="settings">

Replace old endpoints:

````javascript
// Old
const baseURL = 'https://api.example.com';

// New
const baseURL = 'https://api.example.com/v1';
````

  </Step>
  <Step title="Test Integration" icon="play">

Run your test suite and verify bookings at `https://dashboard.example.com`.

  </Step>
</Steps>
