## Overview

Lawnstarter provides a comprehensive SaaS platform designed to streamline how you organize and manage project documentation. You can create structured knowledge bases, collaborate in real-time with your team, and ensure all important information stays accessible and up-to-date. With intuitive tools for versioning, search, and integrations, Lawnstarter eliminates the chaos of scattered docs and scattered files.

The platform supports teams of all sizes, from startups to enterprises, by offering customizable workspaces tailored to your workflows. Focus on your projects while Lawnstarter handles the documentation heavy lifting.

## Quick Start

Get up and running with Lawnstarter in minutes using these steps.

<Steps>

<Step title="Create Account" icon="user-plus">

  Visit [https://app.lawnstarter.com](https://app.lawnstarter.com) and sign up with your email.

  ```bash
  # No code needed - just click "Sign Up"
  ```

</Step>

<Step title="Set Up Workspace" icon="settings">

  After login, create a new workspace for your project.

  <CodeGroup tabs="JavaScript,Python">

  ```javascript
  // Example: Fetch workspace creation via API
  const response = await fetch('https://api.example.com/workspaces', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
    body: JSON.stringify({ name: 'My Project Docs' })
  });
  ```

  ```python
  # Example: Create workspace with requests
  import requests
  response = requests.post('https://api.example.com/workspaces',
                          headers={'Authorization': 'Bearer YOUR_API_KEY'},
                          json={'name': 'My Project Docs'})
  ```

  </CodeGroup>

</Step>

<Step title="Add Your First Document" icon="file-plus">

  Navigate to your workspace dashboard and click "New Page". Start editing with rich text, embeds, or code blocks.

</Step>

</Steps>
