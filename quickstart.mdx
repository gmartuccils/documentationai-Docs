Before starting, ensure you have:
- A valid email address for account verification
- A modern web browser like Chrome, Firefox, or Safari
- `<5` minutes of time
</Callout>

## Create Your Lawnstarter Account

Follow these steps to sign up, log in, and set up your first documentation project.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Navigate to the [Lawnstarter signup page](https://app.lawnstarter.com/signup).

    Fill out the registration form with your email, name, and password. Lawnstarter sends a verification email immediately.

    ```
    Example fields:
    - Email: your.email@example.com
    - Name: Your Full Name
    - Password: (strong password recommended)
    ```
  </Step>

  <Step title="Verify Email and Log In" icon="mail">
    Check your inbox (and spam folder) for the verification email from Lawnstarter.

    Click the verification link, then log in at [https://app.lawnstarter.com/login](https://app.lawnstarter.com/login).

    <Callout kind="tip">
      Enable two-factor authentication during first login for added security.
    </Callout>
  </Step>

  <Step title="Create a New Documentation Project" icon="plus">
    Once logged in, click **New Space** in the top-right dashboard.

    Enter a project name like `My First Docs` and select a blank template.

    Click **Create** to generate your empty documentation workspace.
  </Step>
</Steps>

## Import Initial Content

Populate your space quickly using these methods.

<Tabs>
  <Tab title="Manual Paste" icon="clipboard">
    Open a new page and paste markdown content.

````markdown
# Welcome to My Docs

This is your first page.

## Quick Features

- Real-time editing
- Custom themes
- API integrations
````
  </Tab>

  <Tab title="GitHub Import" icon="github">
    In the workspace settings, connect your GitHub repo.

    Select the branch and import path. Lawnstarter syncs changes automatically.
  </Tab>

  <Tab title="API Import" icon="code">
    Use the Lawnstarter API to bulk import.

<CodeGroup tabs="JavaScript,cURL">
```javascript
const response = await fetch('https://api.lawnstarter.com/v1/spaces/{SPACE_ID}/pages', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    path: 'introduction.mdx',
    content: '# Imported Page\nYour content here.'
  })
});
```
```bash
curl -X POST https://api.lawnstarter.com/v1/spaces/{SPACE_ID}/pages \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "path": "introduction.mdx",
    "content": "# Imported Page\nYour content here."
  }'
```
</CodeGroup>
  </Tab>
</Tabs>

<ParamField path="SPACE_ID" param-type="string" required="true">
  Your documentation space identifier from the dashboard URL.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token obtained from account settings.
</ParamField>
